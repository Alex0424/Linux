# GUI

## Desktop Environment

### List

1. [GNOME](https://www.gnome.org/) (GNOME 49 and higher)
    - very common
    - simple & easy
2. [MATE](https://mate-desktop.org/)
    - fork from GNOME 2 with independent development
3. [KDE Plasma](https://kde.org/plasma-desktop/)
    - lightweight, customizable
4. [XFCE](https://www.xfce.org/)
    - extremely lightweight
5. [Budgie](https://buddiesofbudgie.org/)
    - luxurious home computing experience
6. [Pantheon](https://wiki.archlinux.org/title/Pantheon)
    - similar to MacOS design
7. [Cinnamon](https://github.com/linuxmint/Cinnamon)
    - dev by Linux Mint
8. [Unity](https://unityd.org/)
    - once Ubuntu's default desktop, now comunity-maintained
